## Development

- Watch source files and build on change
- `gulp watch`
- Start development http server (browser-sync)
- `gulp serve`

## Build

Index.html file builds are done automatically on change with the above gulp tasks.

- Manual build (zsh)
- `node ./resume.js >! index.html`
- Gulp Task
- `gulp build`
